Upload API Documentation

This API allows you to upload a file to a specified path within the /www directory.

API Endpoint

/cgi-bin/upload_api.cgi

Parameters

  • file (required): The file to be uploaded.
  • path (required): The absolute path where the file should be saved (starting from /www).

Example Request

curl -F "file=@/path/to/your/file.txt" -F "path=/www/uploaded_file.txt" http://your-server/cgi-bin/upload_api.cgi

Example Request with Basic Authentication

curl -u admin:password -F "file=@/path/to/your/file.txt" -F "path=/www/uploaded_file.txt" http://your-server/cgi-bin/upload_api.cgi

Response

The response will include a log of the upload process.
